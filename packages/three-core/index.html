📦 构建和发布

使用以下命令进行构建和发布：

# 开发模式
npm run dev

# 构建库
npm run build

# 预览构建结果
npm run preview

# 运行测试
npm run test

# 运行测试覆盖率
npm run test:coverage

# 代码检查
npm run lint

# 类型检查
npm run type-check
